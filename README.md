# Traffic Light System Using Arduino UNO

## Project Overview
This project simulates a real world traffic light control system using Arduino UNO and LEDs. the LEDs glow sequentially to represent yellow, green and red signals.

## Components Used
- Arduino UNO
- Red LED
- Yellow LED
- Green LED
- Breadboard
- Jumper Wires
- 220Ω resistors

  ## Working Principle
  The traffic lights operate in a timed sequence:
  1. Yellow LED blinks for 5 times in 10 seconds
  2. Green LED turns ON for 10 seconds
  3. Red LED turns ON for 10 seconds
  4. Sequence repeats continuously
 
  ## Arduino Code
  The Arduino progra controls LED timing using digital  output pins and delay functions.
